# HeadInjectScript
a component react that inject script tag on head tag

# Usage 

npm install react-headinjectscript

```javascript
<HeadInjectScript 
    injectBeforeEvent={callback_before_event}
    injectsource={callback_get_string_src}
    injectDoneEvent{callback_done_event}
 />
```

# Test
npm test

# Run Demo 

cd demo 

```javascript
npm install
```

Run demo 
```javascript
npm start
```
